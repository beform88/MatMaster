# 不锈钢成分强度分析示例问题

## 问题设计

### 主问题（推荐）

**"请帮我调研一下在海洋大气环境中具有良好稳定性的不锈钢成分设计，然后基于一个典型成分（如316L或2205双相不锈钢），系统性地微调铬（Cr）元素的含量（例如从16%到22%，每次增加1-2%），预测不同Cr含量下的抗拉强度，并分析Cr含量与抗拉强度的关系趋势。"**

### 问题分解说明

这个问题可以引导MatMaster agent执行以下步骤：

1. **文献调研阶段**（使用 STEELkb_agent）
   - 查询"耐海洋大气腐蚀的不锈钢成分设计"
   - 查询"316L或2205双相不锈钢的成分和性能"
   - 获取典型的不锈钢成分范围

2. **成分选择阶段**
   - 基于文献调研结果，选择一个基础成分
   - 例如：316L (Fe, Cr~17%, Ni~12%, Mo~2.5%, Mn~2%, Si~0.75%, C~0.03%)

3. **成分微调与预测阶段**（使用 STEEL_PREDICT_agent）
   - 保持其他元素不变，系统性地改变Cr含量
   - 预测每个成分的抗拉强度
   - 例如：
     - Cr16%: 预测强度
     - Cr18%: 预测强度
     - Cr20%: 预测强度
     - Cr22%: 预测强度

4. **关系分析阶段**
   - 整理Cr含量与预测强度的数据
   - 分析趋势关系
   - 解释可能的机理

## 注意事项

1. **成分格式**：预测工具需要特定格式，如 "Fe70Cr17Ni12Mo2.5" 或 "C0.03Si0.75Mn2.0Cr17.0Ni12.0Mo2.5"
2. **元素总和**：需要确保所有元素含量加起来合理（通常主要元素总和接近100%）
3. **元素范围**：确保微调的元素含量在合理范围内
4. **其他元素**：微调一个元素时，需要明确其他元素如何保持（固定值或按比例调整）
